# Cálculos Matemáticos em Python 🧮

Coleção de algoritmos de cálculos matemáticos desenvolvidos em Python durante os estudos de **Bacharelado em Sistemas de Informação** no IFNMG.

## 📚 Sobre o repositório

Este repositório reúne exercícios de cálculos matemáticos diversos: geometria (áreas, volumes), conversões, fórmulas e operações numéricas. Os algoritmos foram desenvolvidos como parte das atividades da disciplina de lógica de programação.

## 📁 Arquivos

| Arquivo | Descrição |
|---|---|
| `ano_bissexto.py` | Verifica se um ano é bissexto |
| `calcula_altura_e_largura_de_um_retangulo.py` | Calcula a área de um retângulo |
| `Calcular_area_de_trapezio.py` | Calcula a área de um trapézio |
| `calculo_raiz_quadrada.py` | Calcula raiz quadrada usando `math.sqrt` |
| `raiz_quadrada.py` | Calcula raiz quadrada com `math.pow` |
| `Consumo_do_veiculo_km_por_litro.py` | Calcula o consumo de combustível (km/l) e custo |
| `Valor_por_kg_de_comida.py` | Calcula o valor total de uma refeição por peso |
| `Verificar_valores_de_tres_triangulos.py` | Verifica se três valores formam um triângulo |
| `Numero_decimal_com_casas_variadas.py` | Formata números decimais com diferentes casas |

## 🛠️ Tecnologias

- **Python 3.x**
- Biblioteca padrão `math`

## ▶️ Como executar

```bash
# Clone o repositório
git clone https://github.com/angeloantunesdarocha/calculos-matematicos-python.git

# Entre na pasta
cd calculos-matematicos-python

# Execute qualquer arquivo
python3 ano_bissexto.py
```
